## 1. 原则1 ｜能力圈（资格）— **结论：擦边圈外**

### 1.1 5 句话讲清生意（独立完成）

1. **生意本质：** Roblox 是「UGC 虚拟世界平台 + 内部经济（Robux）」——平台抽水、把现金返给开发者（DevEx），自己赚 take rate；用户买 Robux 在 user-generated games 里消费 (avatar、道具、解锁)。**财务模型 ≈ 游戏版 App Store / YouTube 混合体**：自营基础设施 + 平台分成。
2. **收入引擎：** 13–15% 的 take rate（开发者拿大头），bookings 递延按 paying user lifetime 摊销成 revenue。Q1 2026: 1.4% of 132M DAU 是 payer（≈1.9M paying user/day），ARPDAU $0.15，ARPU(payer) $10.17/day。
3. **用户结构：** 73% 用户 < 18 岁（按 May 2026 数据 ≈35% 在 <13，38% 在 13–17），核心用户是 9–14 岁的孩子；18+ 是增长的新引擎（U.S. 18+ DAU +40% YoY，18–34 +50% YoY，但 18+ 只占 age-checked DAU 的 26%）。
4. **生意 jaws：** 收入端 +39% YoY，cost 端 R&D + 信任安全 capex 永远在膨胀（Q1 净亏 $246M），靠 SBC 当工资。FCF 看似漂亮但本质是 SBC 摊销 + 应收款时间错位（bookings 比 revenue 早）。
5. **核心风险锚：** 7+ 州 AG 起诉 / 联邦 140+ 集体诉讼 / FTC 调查申请 / Australia eSafety 监管线 —— 全部围绕同一句话："Roblox 给儿童的不是安全的环境"。**这是生意的存在性风险，不是估值风险。**

→ 5 句话能讲清生意结构。**但 test 1 只测了 "现在的样子"。**

### 1.2 Test 2 — Predict downside floor（**这里挂科**）

按 CLAUDE.md 定义："懂 = 能对这门生意的未来做出独立、可证伪的预判"，关键预判要素：

- **壁垒会不会变？** Roblox 的护城河是双边网络（创作者 ↔ 9–14 岁用户）+ 切换成本（Robux 余额、好友图）。但当核心用户群（<13）是监管箭靶时，护城河变成枷锁——你越大、越难甩。
- **Bear floor 我能不能独立画？** 
  - 极端 bear：U.S. 跟随 Australia 模式硬性 <16 social-media 限制 → 一刀切砍 ~50–60% bookings。EV/Rev 跌到 hyperscaler 受罚后估值（~2x rev）→ market cap ~$8–10B → 股价 **$11–14**（从今天 –70%）。
  - 温和 bear：safety 摩擦持续 4–6 quarter，bookings growth 持续在 5–10% 而非加速回 20%+，DAU Q3 不反弹 → 共识下修，倍数压到 EV/Rev 2.5x → 股价 **$22–25**（从今天 –42%）。
  - 中性 bear："Q3 sequential DAU 反弹"如期出现，guide 中点兑现，FY27 bookings 回 15–18% → 股价 **$38–42** 持稳。
- **问题：监管 bear 的概率分布我没有 edge。** 这不是技术或 take rate 题，是政治/法律题。我不知道 FTC 调查会立案、几率多少；不知道 Florida/Texas 案件最终判决会怎样；不知道 Roblox Kids/Select 6 月上线能不能堵住 PR 漏洞。**这些是必须有"独立预判"才有资格独立估胜率的变量，我没有。**

### 1.3 Test 3 — 跌 50% 敢加吗

从 $42 → $21，敢不敢加？答案：**不敢全力加。**
- $21 大约对应 EV/Rev 2x（FY26 $7.5B 收入），这个倍数下要么是监管事件已落地（FTC 立案 / 多州集体和解），要么是 bookings guide 进一步下修。**我无法独立判断 $21 时是「机会」还是「半山腰」。** 没有独立可证伪预判 → 没有资格加。

### 1.4 资格判定

| Test | Pass? | 备注 |
|---|---|---|
| 5 句话讲清生意 | ✅ | 平台 / take rate / 用户漏斗清晰 |
| Predict downside floor | ❌ | floor 取决于监管路径，我没 edge |
| 跌 50% 敢加 | ❌ | $21 是机会还是 trap 我不知道 |

**判决：原则1 不过关。属于 "可补习但仍在补习期" → 表达方式应是 ETF（QQQ/视频游戏 ETF 如 HERO/ESPO），不开单股集中仓。**

> 框架原文："圈外标的没资格估胜率。" 后面原则2/3 仍跑，但只是 sanity check，不是开仓许可。

## 2. 原则2 ｜值不值得 + 攻守归类

### 2.1 看壁垒（先回答 "好生意吗"）

**护城河四件套：**
1. **双边网络效应（核心）：** 创作者去用户多的平台 → 用户去内容多的平台。1300 万创作者 + 132M DAU，规模上 Fortnite/Minecraft 都打不进来。
2. **Robux 经济体闭环：** Robux 是 platform-bound 货币，余额绑死用户 + 给开发者 DevEx 形成依赖。
3. **基础设施（Roblox Cloud）：** "<$0.01/hour" 单位经济（管理层数据）+ 实时多人引擎，第三方 publisher 复制成本高。
4. **(新增) 安全合规壁垒：** 管理层把"我们是唯一在认真做 <13 安全的大平台"包装成 moat。**但这恰恰也是负债：未来 3 年合规成本只会上升。**

**壁垒会不会变（独立判断）：**
- **被侵蚀方向：** Fortnite 创意模式（UEFN）/ Minecraft 在抢 9–14 岁用户的部分时长；但短期看 Roblox 在 UGC 集成度 + 创作者经济上仍领先 2–3 年。
- **强化方向：** 18+ cohort 增长（U.S. 18–34 +50% YoY）是真实的，如果 5–10 年 RBLX 能把用户结构从 "60% 儿童 / 40% 青少年+成人" 翻成 "30/70"，监管风险结构性下降。但这需要 2–3 年新游戏类型（社交、约会、虚拟工作）成立。

**判定：是好生意但不是 "未来三年护城河变深" 的生意——是 "护城河被监管侵蚀压力 vs. 18+ 自然增长" 的拉锯。**

### 2.2 四维度

| 维度 | 是否满足 | 证据 / 反驳 |
|---|---|---|
| ① 高增长 | **半满足** | Q1 +43% bookings 是高增长 ✓；但 FY26 guide 只 +8–12% bookings（cut from +22–26%），增长直接打骨折 ✗。看 multi-year base case 而非 next 12m → 长期 15–20% bookings CAGR 仍可能成立，但需 Q3 反弹证伪 |
| ② 极致稳定+避险 | **不满足** | 监管 / 法律风险是存在性级别；7 州 AG + FTC 申请 + 140+ 联邦集体诉讼。这是 anti-stable |
| ③ 股东回报 | **刚刚满足** | 首次 $3B 回购授权（May 2026）—— 但本质是 **抵消 SBC 摊薄**（FY25 SBC $1.13B）。Net buyback = 几乎为零。不算真股东返还 |
| ④ 想象力 | **满足** | Roblox Reality（photoreal 多人）、immersive ads、18+ 经济、虚拟商品 + 虚拟工作场景。CEO 反复讲 "10% of gaming"（目前 ~3%）的长 TAM。**但想象力 = 远期，不抵 12 个月监管现实** |

**攻守分组：**
- 进攻型门槛：① 或 ④ 任一 → ④ 满足 → **算进攻型**
- 防御型门槛：② 且 ③ 都要 → ② 不满足 → **不是防御型**

→ **进攻型候选**。

### 2.3 共识 vs 我

| 项 | 共识叙事 | 我的判断 |
|---|---|---|
| Bookings deceleration 是暂时的 | 是，Q3 见底反弹 | 中性。管理层信心来自 "Q3 季节性 + 6 月 Kids/Select 上线"，但 Kids/Select 的影响要 2–3 个 quarter 才能数据化 |
| Safety = compounding moat（mgmt 话术） | 部分接受 | 短期是反向 moat（friction），长期才可能正向 |
| 18+ cohort 是新引擎 | yes | yes，但 26% 占比、绝对值还小 |
| FTC 立案概率 | 共识近乎不计入 | **我会给 15–25% 概率**，对应 –30% 估值冲击 |
| TAM "10% of gaming" 现实性 | 卖方画 5 年 | 5 年内若监管不松，到不了；3% → 5% 是合理 base |

**与 Wall Street 最大 delta：监管/法律 tail risk 是否应该入价。** 共识 PT $64.81 隐含 12m 上行 ~52%，对应 EV/Rev 5.5–6x FY27——只有在 "监管阴影逐步消散 + bookings growth 重新加速到 20%" 的乐观路径下才合理。

## 3. 原则3 ｜胜率 × 赔率（仓位）

### 3.1 拆解

**胜率 = base case (Q3 sequential DAU 反弹 + FY27 bookings growth 重回 18%+ + 无重大监管裁定) 真兑现的概率：**
- Q3 DAU 反弹：管理层语气有信心（"largely driven by return to DAU growth, not heroic things in back half"），但 last quarter 他们也 "have confidence"。给 **55%**。
- 监管不出黑天鹅（未来 24 个月无 FTC consent decree / 无 U.S. 联邦立法仿 Australia under-16 ban / 无 major state 判决 > $500M）：给 **70%**。
- 两者条件独立近似 → **base case 胜率 ≈ 55% × 70% = 38%**。

**赔率 = 2 年 base case 上行 ÷ bear 下行：**
- Bear case 下限（综合 floor 见 1.2）：~$22（温和 bear, –48%）。如果 catastrophic bear 触发 → $12–14；权重平均 ~$20。
- Base case 2 年目标价：假设 FY27 bookings $9.5B（+15% from FY26 guide 中点 $8.25B），EV/Bookings 4x（合理给 deceler 后的 high-growth 平台），EV ~$38B，stock **$53**。
- Bull case：FY27 bookings $10.5B（重新加速到 18%），EV/Bookings 5.5x（共识恢复），stock **$80**。
- 加权 2 年目标 = 0.38 × $53 + 0.30 × $80 + 0.32 × $22 = **$52.5**。
- 赔率 = $52.5 / $22 ≈ **2.4x**。
- **Base 单情景**(单单看 base path) 2 年 base / bear = $53 / $22 ≈ **2.4x**（不到 2x 门槛要求的"显著高于"）。

### 3.2 仓位档位决策

CLAUDE.md 进攻型大仓门槛：「下限明确 **且** 2 年 base case ≥ 2x（**都要**）」。
- 下限明确？❌ —— 监管 floor 不可控，从 $22 到 $14 都可能
- 2 年 base ≥ 2x？✓ 勉强（base $53 / 现价 $42.5 = 1.25x；只有 vs bear floor 才 2.4x，但 anchor 应是 vs **现价**看 base 兑现的回报。**Base 兑现仅 +25% over 2 years，不到 2x 门槛**）

→ **够不到进攻型大仓门槛**

→ CLAUDE.md 明文："进攻但够不到 2x → 不开单股，买 QQQ"。

是否进入 "低胜率高赔率 → 小仓 5%" 桶？
- 胜率 38% 不算低
- 赔率 vs bear 2.4x 算中等
- 这桶通常是 "10–20% 概率 + 3–5x" 的彩票型，RBLX 不符合
→ **也不是低胜率高赔率彩票仓**

### 3.3 最终仓位结论

| 选项 | 判定 |
|---|---|
| 防御型大仓 | ❌ 不是防御型 |
| 进攻型大仓 | ❌ 原则1 没过关 + base case 上行 < 2x |
| 小仓（彩票） | ❌ 胜率不够低、赔率不够高，不符合该桶画像 |
| **建仓决定** | **不建仓** |
| 替代方案 | **QQQ**（赚指数 β），或 ETF HERO/ESPO 接住游戏行业 β 而不承担单股 idiosync risk |
