5句话看懂阿里当前的状态

1. 战略重新统一：吴泳铭2024年中收回"6+N"分拆方案，确立"AI+电商"双引擎，现金流由淘天哺育云和AIDC，组织合并大幅减少内耗。

2. 云的二次起飞：从拼多多冲击下挣扎多年的"边缘业务"重回主舞台 — 36%同比增速接近2018年水平，AI产品三位数增长持续10个季度，Qwen模型已成中文模型事实标准之一。

3. 电商守势但不溃败：CMR基本平稳，但"淘宝闪购"补贴战让中国电商集团调整后EBITA同比塌陷43%，本季度光是即时零售就烧掉 ~¥50bn 补贴。

4. 资本承诺空前：3年¥3,800亿(~$525亿)AI/云基建投入，CFO在Q3电话会明确表示"380亿这个数字可能偏小"，跟随北美超大规模厂商节奏。

5. 股东回报底：18个月回购缩减股本约6%，FY26 派息 HK$1.96 (含特别股息)，估值低位下回购等同最佳capital allocation。

用户视角的关键问题：你买的不是"PE便宜的中国电商"，而是"被低估的Hyperscaler+被定价为零的Qwen+稳态淘天现金牛"。如果云未来12个月维持30%+增速且margin扩张，市场会从"按电商P/E定价"切换到"分部估值"（SOTP），股价重估空间约35–60%。

战略关键点 — 你需要理解的3个事

(a) 淘宝闪购 = 战略防御不是新增长极。 美团的Shangou曾威胁淘宝在低线市场和年轻用户的复购心智。阿里2025年砸¥50bn补贴换来3亿月活和单日1.2亿订单。这是用现金保心智，不要期望短期变现，但等补贴战平息(预计2026下半年)，3亿用户存量会回流主站交易。

(b) 云 = 中国对外输出AI能力最完整的玩家。 字节有豆包(自研主力模型，benchmark上跟Qwen互有胜负)，但火山引擎云市场份额仍小且豆包以服务抖音/TikTok内部推荐为主，企业外销刚起步；腾讯云规模较小且混元主要服务自身生态；华为云盘古以行业模型为主+受芯片采购限制。阿里的差异化是「IaaS市场份额第一(~36%) + Qwen开源生态主导地位 + 15年企业客户积累」三位一体 — Qwen开源策略让全球开发者大量基于Qwen微调衍生模型(HuggingFace下载量已超Llama)，这是其它中国厂商没有的网络效应护城河。

(c) 国际(AIDC)将于FY27扭亏。 AliExpress Choice业务UE改善是关键变量，与Temu不同的是阿里有Lazada+Trendyol的本地化壁垒；亏损从去年同期¥4.95bn收窄到¥2.0bn，扭亏点临近。

争论点 1 · 云的估值倍数

多头(JPM/Jefferies)：吴泳铭的"5年$100bn"目标隐含CAGR 25%+，应该按6-8x P/S估值(对标AWS段)。

差异 → ~$40/ADR

争论点 2 · 即时零售何时停战

多头：监管已开始关注无序竞争，2026下半年市场默契形成，FY27减亏一半。

空头(GS)：美团/JD都有现金弹药硬扛，价格战可能延续到FY28。

差异 → ~$25/ADR

争论点 3 · Qwen独立估值

多头(MS高端SOTP)：成立独立"Token业务集团"暗示分拆可能，类比OpenAI估值给$15-20/股。

空头：开源模型变现路径未验证，Qwen API实际收入仍不透明，应已包含在云估值内。

我对卖方观点的看法

卖方共识$189基本与我的$190相符 — 这不是巧合，是因为共识的Bull/Bear三情景加权刚好落在这里。但有几点值得注意：

  • JPM的$245可能过激进。Alex Yao把Qwen独立估值$30+/ADR的逻辑站不住 — Qwen实际API收入还没公开披露，开源策略下变现路径不清晰。把这部分剔掉，JPM的电商+云核心估值大约$200-210，与其他多头一致。
  • GS的$163可能过悲观。GS对即时零售的悲观假设(战争延续到FY28)概率不超过20% — 监管已经在2025年底点名"无序补贴"，市场最早在2026下半年开始默契降速。
  • 真正的alpha来自情景切换。如果你认为现在分歧的3个点中至少2个会朝多头方向解决(云高倍数 + 即时零售FY27减亏)，那$200-220目标价是合理的；如果3个全部朝多头，$240+。
  • 注意目标价集中下调期。Q3财报后(2026/3/18)Jefferies从$225降至$212、Barclays从$190降至$186 — 这是分析师对短期EPS下修的反应，而不是长期thesis变化。这种"目标价小幅下修但维持买入"的pattern通常预示股价6-12个月后的反弹机会。

⑧ 多空论点合成

★ Bull Case

  • 云的二次起飞被严重低估。 AI产品三位数增速持续10季，36%整体增速接近2018年高点；Qwen成为中文模型事实标杆。如果按6x P/S单估，云一项就值 ~$60/ADR。
  • 组织复归提升执行效率。 吴泳铭收回拆分计划，资源集中到"AI+电商"双主轴，2024-2025已经看到决策效率明显提升。
  • 电商竞争格局趋稳。 PDD自身增速从90%坠落到12%，监管对食品/医药品类清退仍利好阿里高质量货盘。淘天CMR持续+5%是稳态信号。
  • $425亿净现金 + 持续回购 = 极强下行保护。 18个月回购缩股6%，按当前节奏FY27前再缩~3-4%；分红已正常化。
  • AIDC接近扭亏。 国际业务亏损同比腰斩，AliExpress Choice UE改善 + Lazada/Trendyol本地化壁垒。
  • 政策端见底。 2024年罚款已结清，2025年起监管重回支持平台经济基调。

✗ Bear Case

  • 即时零售可能是无底洞。 美团/京东/阿里三方都有现金弹药，价格战可能延续到2027；本季已烧¥50bn换3亿用户，留存度成疑。
  • Capex回报率有重大不确定性。 $525亿三年投入是历史最大豪赌，如果云需求增速2027放缓而设备折旧到位，short-term margin再压缩。
  • 地缘政治尾部风险。 中美科技脱钩升级 / 台海 / 美国对华芯片限制(影响阿里云训练算力)都是显著tail risk。
  • 淘天份额持续被PDD和抖音蚕食。 虽然增速企稳，但绝对GMV份额从2020年的~60%已降到~45%，这个长期下行趋势没有反转。
  • Qwen商业化路径未证伪。 开源模型的tokens变现能力低于OpenAI/Anthropic封闭路径，Qwen独立估值的故事还没数据支撑。
  • Holdco折扣可能扩大。 一旦市场focus回到"中国监管+复杂结构"，10-20%折扣可能变25%。

我的判断

这是一个 "右尾大、左尾有限" 的非对称机会。下行有$425亿净现金和持续回购铺底(隐含底部$110-120左右);上行靠云重估 + 即时零售烧钱周期结束 + AIDC扭亏三个独立催化剂。不是PDD那种"超低估值+高速增长"的简单价值股，而是"复杂业务组合+隐藏资产(云/Qwen/Ant)被低估"的SOTP机会。

建仓节奏建议：

  • 第一档(40%)：当前价位 ~$140 附近建仓 — 已经在年线和上升通道下沿。
  • 第二档(30%)：$125附近 — 如果即时零售战恶化或Q4 FY26财报再miss，市场会过度反应。
  • 第三档(30%)：$110-115 — bear case价位，需要bear触发器(地缘/监管)，但这正是建仓最好的时点。

持仓时间：18-24个月。关键观察点(catalysts)：
